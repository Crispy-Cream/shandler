# Shandler

## Installing Shandler

To install Shandler with NPM, run the following.

```text
$ npm install shandler@latest
```

{% hint style="info" %}
To download Shandler's Beta, run `npm install shandler@dev`
{% endhint %}

To install Shandler with yarn, run the following.

```text
$ yarn add shandler
```
